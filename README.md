A API fornece todas as operações básicas de um CRUD (Create, Read, Update e Delete),
Toda a comunicação entre front-end e back-end é feita via requisições HTTP (JSON),
o back-end é consumido por uma interface de cadastro desenvolvida em React, onde pode-se:

    -Preencher formulário de cadastro
    -Visualizar a lista de usuários cadastrados
    -Atualizar informações
    -Remover usuários

Tecnologias Utilizadas:

    .Node.js
    .Express
    .Prisma ORM
    .Mongo DB
    .CORS
    .React
